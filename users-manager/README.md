## Getting Started

- Clone repo

```
git clone git@gitlab.asoft-python.com:van.nguyenthi/react-training.git
```

- Checkout branch

```
cd users-manager
```

- Install dependencies

```
pnpm install
```

- Run Webpage

```
pnpm dev
```

Open http://localhost:3000/ in browser


- Run Storybook

```
pnpm storybook
```

Open http://localhost:6006/ in browser
